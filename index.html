Testing de Software
En el proceso de desarrollo de software es normal encontrar errores.
Cuando esto sucede en la etapa de prueba de software, no supone un gran inconveniente.
Continuar sin abordarlos puede generar problemas graves para todas las partes involucradas en el proceso de desarrollo del proyecto.
El testing de software juega un papel fundamental y supone una garantía de calidad de suma importancia para cualquier empresa.
